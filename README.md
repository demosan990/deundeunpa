# 정이든 마케팅 웹사이트

AI와 자동화로 비즈니스 성장을 돕는 정이든 마케팅의 공식 웹사이트입니다.

## 🌐 서비스
- CRM 구축
- AI와 바이브코딩 교육
- 온라인 BM 컨설팅
- 온라인 마케팅 대행

## 🎯 타겟 고객
- 레슨 및 코칭 (음악, 체육, 학과목 강사)
- 상담가 (사주, 심리 상담 등)
- 소상공인
- 영업직

## 🛠️ 기술 스택
- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- Vanilla JavaScript
- localStorage (임시 데이터 저장)

## 📁 프로젝트 구조
```
├── index.html          # 메인 페이지
├── portfolio.html      # 포트폴리오 페이지
├── services.html       # 서비스 상세 페이지
├── contact.html        # 문의하기 페이지
├── admin.html          # 관리자 페이지
├── css/
│   └── style.css       # 디자인 시스템 및 스타일
├── js/
│   ├── data.js         # 데이터 관리
│   └── main.js         # 공통 기능
└── assets/
    └── images/         # 이미지 파일
```

## 🚀 로컬 실행
```bash
npx serve . -p 3000
```
